# JavaScript Advanced

### About the course

In the "JavaScript Advanced" course, students gain in-depth knowledge of the JavaScript language such as: basics of syntax, working with arrays, matrices, objects, classes and writing functions. They also learn more complex concepts such as: function context, explicit binding, event loop. The course develops algorithmic thinking. Upon successful completion of this course, students have the opportunity to work with the DOM tree, manipulate the DOM tree and work with "events". Functional and OOP approaches to programming with JavaScript are explored, and concepts such as: inheritance, object composition and prototype chain are studied.

### Skills acquired after successful completion of the course

- Working with classes and objects
- Working with functions and JSON formats
- Working with linear structures
- Retrieving information and manipulating the DOM tree
- Implementing object-oriented programming
- Implementing functional programming
